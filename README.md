# Financial Performance Analysis with SQL

## Project Overview
This project analyzes financial performance across different countries using SQL. The goal is to understand how much revenue each market generates and how profitable each country is after considering marketing costs.

## Business Objective
The analysis was designed to answer two key business questions:

- How much are we earning by country?
- How profitable is each market when marketing expenses are considered?

## Dataset
The project is based on financial and marketing data combined into a working table with the following business metrics:

- Revenue
- Costs
- Margin
- ROI

## Key Metrics Analyzed
- Revenue by country
- Total costs by market
- Margin performance
- Return on Investment (ROI)
- Market-level profitability comparison

## What Was Done
- Queried financial data using SQL
- Calculated core KPIs such as revenue, cost, margin, and ROI
- Compared financial performance across countries
- Identified more profitable and less efficient markets
- Summarized findings in an executive business format

## Main Skills Demonstrated
- SQL queries
- Filtering and aggregations
- KPI calculation
- Financial analysis
- Business insight generation
- Executive reporting

## Files Included
- Executive summary in Excel
- Supporting project documentation

## Project Outcome
This analysis helps support financial decision-making by identifying which markets generate stronger returns and which ones may require cost review or marketing optimization.

## Tools Used
- SQL for financial analysis and KPI calculation
- Excel for the final executive summary presentation
